Utilizei como tecnologia:

* Vite,
* React (com TypeScript),
* TailwindCss,

e para pegar as mesma cores da imagem de referencia eu utilizei o site: https://www.figma.com/pt-br/seletor-de-cores/


Utilizei elas pelo fato de estar mais familiarizado, pela estrutura fornecida pelas documentações, pela facilidade de manutenção que a tecnologia nos possibilita, pore ter uma comunidade gigantesca juntamente com a documentação dita anteriormente também, como aquitetura, optei por uma arquitetura limpa orientada por UseCases, mesmo que só tenha 1 useCase no momento, a escolha da arquitetura possibilita um melhor desenvolvimento no caso  de novas implementações nesse mesmo projeto.


para realizar a troca de tema tomei a liberdade de adcionar um simples botão de tema no canto superior direito da tela.


Para o teste é simples, basta dar instalar as dependencias utilizando o terminal:


instalar as dependencias

```bash
npm install
```


e rodar:

```bash
npm run dev
```
