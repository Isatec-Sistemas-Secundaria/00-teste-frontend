Sobre o projeto:

O projeto consiste em uma tela de login, seguindo o layout que foi passado.

O foco foi reproduzir fielmente o design proposto, garantindo uma boa experiência do usuário e similaridade do resultado entregue com a demanda.

Ferramentas utilizadas:

Escolhi usar React para aprender e praticar, ao invés de usar só HTML estático. 

A estilização foi feita com CSS.

O projeto começou com TypeScript mas eu escolhi trocar para o JavaScript durante o desenvolvimento.

Tema claro e escuro: 

A aplicação tem suporte a tema claro e escuro, permitindo melhor adaptação visual para o usuário. Para mudar entre os temas é só clicar no logo em cima do "Bem-vindo".
