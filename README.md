## Sobre as Tecnologias Usadas
Desenvolvi usando HTML, CSS e JavaScript puro, mas com uma pequena diferença, utilizei Scss para a estilziação.

Scss é um pré-processador de CSS que funciona quase de mesma maneira, mas ao contrário do CSS normal, permite um estura de código mais limpa e organizada, além de permitir inserir variáveis no código para facilitar no processo de desenvolvimento. Gosto de usa-lo pois otimiza muito meu tempo na hora de desenvolver, pois me organizo melhor e consigo fazer futuras atualizações com mais facilidade e rapidez.
